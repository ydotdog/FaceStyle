# FaceStyle

iOS 脸型分析与发型推荐 App — 基于 iPhone TrueDepth 摄像头。

## 功能

- **3D 脸型扫描**：使用 TrueDepth 摄像头获取精确的面部三维数据
- **脸型分类**：自动分析脸部比例，识别脸型（圆脸、方脸、椭圆脸等）
- **发型推荐**：根据脸型匹配最适合的发型方案
- **中英双语**：支持中文和英文界面

## 技术栈

- Swift / SwiftUI
- ARKit（TrueDepth 面部追踪）
- Core ML

## 系统要求

- iOS 15+
- 配备 TrueDepth 摄像头的 iPhone（iPhone X 及以上）
- Xcode 15+

## 构建

使用 Xcode 打开 `FaceStyle/FaceStyle.xcodeproj` 并构建运行。
